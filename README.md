Renombra Archivos .EXE y .EDU recursivamente eligiendo una carpeta de windows.

Es una app de windows; la ventana muestra 2 botones. Boton EXE to EDU: Abre ventan de archivos, y se debe seleccionar una carpeta raiz. Al aceptar, la app busca recursivamente desde esa carpeta todos los archivos con extensión .EXE y hace una copia de estos abregando la extensión .EDU

Boton EDU to EXE: Abre ventan de archivos, y se debe seleccionar una carpeta raiz. Al aceptar, la app busca recursivamente desde esa carpeta todos los archivos con extensión .EDU y hace una copia de estos dejando la extensión .EXE
